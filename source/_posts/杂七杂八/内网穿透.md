# 内网穿透方案_ZeroTier
## Liunx 配置

1) 安装

```bash
curl -s https://install.zerotier.com | sudo bash
```

2) 启动并设为开机自启

```bash
sudo systemctl enable --now zerotier-one
sudo systemctl status zerotier-one --no-pager
```

) 加入网络（Network ID）

```bash
sudo zerotier-cli join <NETWORK_ID>
```


4) 去 Central 授权


5) 验证是否拿到 Managed IP

```bash
sudo zerotier-cli status
sudo zerotier-cli listnetworks
ip a | grep -n "zt"
```

## Windows（Win10/Win11）

1) 安装

1. 去 ZeroTier 官网下载 Windows Installer（.msi / .exe）
2. 双击安装（默认一路 Next 即可）

2) 启动方式

* 安装后通常会自动启动服务，并在右下角托盘出现 **ZeroTier** 图标
* 若没看到托盘图标：开始菜单打开 **ZeroTier One**（会拉起托盘）
* 或检查服务：`Win + R` → `services.msc` → 找到 **ZeroTier One**（或类似名字）→ 启动

3) 加入网络（GUI）

1. 右下角托盘 ZeroTier 图标右键
2. **Join New Network…**
3. 粘贴 `<NETWORK_ID>` → Join



## 最小联通测试

在任意一端用对方 Managed IP：

* 测试连通：

```bash
ping <对方Managed_IP>
```

* SSH（Linux 目标）：

```bash
ssh user@<对方Managed_IP>
```

* RDP（Windows 目标，默认 3389）：
  `mstsc /v:<对方Managed_IP>`
